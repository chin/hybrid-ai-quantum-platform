# AI Use Disclosure

Generative AI tools, including ChatGPT by OpenAI, were used as development assistants for OptEngine.

AI assistance included:

- tracking project objectives, goals, milestones, and unresolved work;
- helping preserve process and architectural decisions during refactoring;
- drafting code stubs and scaffolding for review and later implementation;
- identifying potential gaps, inconsistencies, and unclear responsibility boundaries in the object-oriented design;
- drafting and organizing technical documentation and architecture diagrams; and
- improving the formatting, type annotations, naming consistency, and formal structure of Python files.

The project author directed the requirements and architectural intent, reviewed all accepted AI-assisted output, and remains responsible for the source code, mathematical formulations, tests, results, and conclusions. AI-generated suggestions were treated as proposals rather than authoritative results. Placeholder or scaffolded code is not represented as completed functionality, and reported results must come from reproducible project executions.

No confidential, restricted, proprietary third-party, or personally identifiable data was intentionally provided to AI tools.
